# Annamayya Keerthanas Indariki Abhayambu

| Devanagari | English |
| ------ | ------ |
|  |  |
| अन्नमय्य कीर्तन इन्दरिकि अभयम्बु   | annamayya kīrtana indariki abhayambu   |
|  |  |
| रागं: काम्भोजि रागं   | rāgaṃ: kāmbhoji rāgaṃ   |
|  |  |
| इन्दरिकी अभयम्बु लिच्चु चेयि   | indarikī abhayambu lichchu cheyi   |
| कन्दुवगु मञ्चि बङ्गारु चेयि ‖   | kanduvagu mañchi baṅgāru cheyi ‖   |
|  |  |
| वॆललेनि वेदमुलु वॆदिकि तॆच्चिन चेयि   | velaleni vedamulu vediki techchina cheyi   |
| विलुकु गुब्बलि किन्द चेर्चु चेयि ❘   | viluku gubbali kinda cherchu cheyi ❘   |
| कलिकियगु भूकान्त कागलिञ्चिन चेयि   | kalikiyagu bhūkānta kāgaliñchina cheyi   |
| वलवैन कॊनगोल्ल वाडिचेयि ‖   | valavaina konagolla vāḍicheyi ‖   |

| तनिवोक बलि चेत दानमडिगिन चेयि   | tanivoka bali cheta dānamaḍigina cheyi   |

| मॊनसि जलनिधि यम्मुमॊनकु तॆच्चिन चेयि   | monasi jalanidhi yammumonaku techchina cheyi   |

| तिरुवेङ्कटाचल धीशुडै मोक्षम्बु   | tiruveṅkaṭāchala dhīśuḍai mokśhambu   |
| तॆरुवु प्राणुल कॆल्ल तॆलिपॆडि चेयि ‖   | teruvu prāṇula kella telipeḍi cheyi ‖   |
